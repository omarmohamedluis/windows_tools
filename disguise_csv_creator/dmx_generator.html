OmiLED Mapper Guide

Navigation

  • Left Click Drag (on background) Pan Grid
  • Middle Click Drag Pan Grid Anywhere
  • Wheel Vertical Scroll
  • Shift + Wheel Horizontal Scroll
  • Zoom Controls (+/-) Bottom-Left

Shortcuts

  • Arrows Move Selection
  • Ctrl + Arrows Multi-Select (Add)
  • Shift + Click Range Select (Directional)
  • Ctrl + Click Toggle Selection
  • Ctrl + A Select All
  • Esc Clear Selection
  • Ctrl + Z Undo
  • Ctrl + Shift + Z Redo

Workflow

  1. Define Grid: Set Width/Height. (Updates don't destroy patches).
  2. Select Pixels: Use mouse or keyboard.
  3. Patch: Set Start Universe/Channel. Choose Order. Click APPLY PATCH.
  4. Review: Check specific pixels in the legend below sidebar.
  5. Export: Save as CSV for Disguise.
